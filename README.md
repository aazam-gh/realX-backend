# realX Backend

This repository is the only production deployment owner for the
`reelx-backend` Firebase project:

- First-party Cloud Functions
- Firestore rules and indexes
- Storage rules
- Production recovery and deployment configuration

Run `npm run check` before any production deployment. See
`docs/PRODUCTION_DEPLOYMENT.md` for guarded deployment and recovery procedures.

The `realx-dev` Firebase project has not yet migrated to this repository.
